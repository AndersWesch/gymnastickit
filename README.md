# Gymnastick It #

This is the best and only place to play your gymnastic stick it games.

### Site ###
www.gymnastickit.com

### How do I get set up? ###

* Copy project
* Run: `composer serve` or `php -S 127.0.0.1:8000 -t public/`
* Open brower on http://127.0.0.1:8000/
